# FlytBase UAV Strategic Deconfliction System

## Overview
This repository implements a UAV Strategic Deconfliction System for FlytBase Robotics Assignment 2025.  
It checks drone missions for spatial and temporal conflicts in shared airspace.

## Folder Structure
- `src/` : Python code
- `docs/` : PDFs
- `visuals/` : Optional plots/screenshots
